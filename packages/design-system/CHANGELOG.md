# Changelog

All notable changes to HAPAI Design System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-07

### Added
- Initial release of HAPAI Design System
- Tailwind 4 integration
- Color tokens (primary, secondary, neutral, semantic)
- Typography scale (headings, body, custom sizes)
- Spacing scale (4px base unit)
- Button component with variants (primary, secondary, danger, ghost) and sizes (sm, md, lg)
- Storybook integration for component documentation
- TypeScript support with strict mode
- ESLint and Prettier configuration
- Vitest for unit tests

### Components Included
- Button
  - Variants: primary, secondary, danger, ghost
  - Sizes: sm, md, lg
  - States: disabled, active, hover

### Design Tokens
- Colors: 10 shades per primary/secondary/neutral + semantic colors
- Typography: 8 font sizes + custom heading/body scales
- Spacing: 32 scale values from 0px to 384px
